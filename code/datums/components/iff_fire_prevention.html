/datum/component/iff_fire_preventionA component that prevents gun (although you can attach it to anything else that shoot projectiles) from shooting when mob from the same faction stands in the way. You can also pass number of ticks, to make gun have an additional delay if firing prevention comes into play, but it is not neccesary.
